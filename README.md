## Описание проекта

CollectTheCube — это интерактивное Android-приложение для поэтапного обучения сборке классического кубика Рубика 3×3. Приложение предоставляет структурированный курс из 7 этапов с возможностью отслеживания прогресса и визуализацией статистики обучения.

## Функционал

### **Аутентификация и безопасность**
- Регистрация новых пользователей
- Авторизация существующих пользователей
- Валидация данных (логин от 3 символов, пароль от 6 символов)

### **Система обучения**
- **Язык вращений** с изучением стандартных обозначений поворотов
- **7 этапов сборки** с подробными инструкциями:
  - Крест на верхней грани
  - Углы первого слоя
  - Ребра среднего слоя
  - Крест на нижней грани
  - Ориентация ребер нижнего креста
  - Расстановка углов нижнего слоя
  - Ориентация углов последнего слоя

### **Отслеживание прогресса**
- Отметка этапов как изученных
- Автоматическое сохранение прогресса
- Визуальная индикация пройденных этапов

### **Статистика обучения**
- **Недельная статистика активности**
- Визуализация в виде столбчатой диаграммы
- Отслеживание количества изученных этапов по дням
- Автоматическое обновление данных

## Архитектура

### **Клиентская часть (Android)**
- **Язык**: Kotlin
- **Минимальная версия SDK**: API 24 (Android 7.0)
- **Архитектура**: MVC (Model-View-Controller)
- **Основные активности**:
  - `RegisterActivity.kt` — регистрация и авторизация
  - `CatalogActivity.kt` — каталог этапов обучения и статистика
  - `RotationGuideActivity.kt` — обучение языку вращений
  - `Stage1Activity.kt` — этапы обучения (1-7)
- **Вспомогательные классы**:
  - `ContentLoader.kt` — загрузчик контента из Firebase
  - `StatisticManager.kt` — менеджер статистики обучения

### **Серверная часть (Firebase)**
- **Firestore** — хранение данных пользователей и контента
- **Структура данных**:
  - Коллекция `users` — информация о пользователях
  - Коллекция `stages_content` — учебные материалы этапов

### **Класс ContentLoader**
Отвечает за загрузку учебных материалов из Firebase Firestore. Поддерживает динамическую загрузку контента этапов в формате JSON с преобразованием в структуры данных Kotlin.

### **Класс StatisticManager**
Управляет сбором и обработкой статистики обучения. Реализует:
- Запись учебных сессий
- Агрегацию данных за 7 дней
- Формирование данных для визуализации в виде графика
- Обработку JSON-данных статистики

**Особенности реализации StatisticManager:**
- Хранение статистики в формате JSON
- Автоматическое ограничение данных 7 днями
- Поддержка уникальных этапов в сессии
- Форматирование дат для отображения на графике
- Обработка ошибок и восстановление данных

## Установка

### Шаги установки
1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/Andrikck/CollectTheCube.git
   ```

2. **Настройка подключения к Firebase**
   - Приложение уже настроено для подключения к существующему Firebase проекту
   - Все данные хранятся в облаке и синхронизируются автоматически

Для ознакомления с функционалом можно использовать:
